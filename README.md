# Trader Behavior vs Market Sentiment Analysis

This repository contains my submission for the Data Science assignment focused on
analyzing trader behavior under different Bitcoin market sentiment conditions.

# Project Structure:
- notebook_1.ipynb: Main Google Colab notebook containing data cleaning, analysis,
  visualization, and statistical testing.
- csv_files/: Contains processed and intermediate CSV data files used in the analysis.
- outputs/: Contains visual outputs such as charts and graphs generated during EDA.
- ds_report.pdf: Summary report highlighting key insights, conclusions, and
  trading recommendations.

# Notes:
- All analysis was performed in Google Colab.
- The notebook is intended to be run top-to-bottom for full reproducibility.
- Google Colab access is set to "Anyone with the link can view".
